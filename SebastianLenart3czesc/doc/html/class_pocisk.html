Opis szczegółowy

The Pocisk class Klasa podstawowa zawierajaca wspolne cechy pociskow.

Parametry
Pocisk- konstruktor uruchamia timer ktory co jakis czas uruchamia metode move
ustawGrafike- wirtualna metoda, ktorsa ustawia grafiki w klasach nizej
setPointCenterObiekt- zapisuje punkt srodka grafiki/obiektu
getPktSrodek- zwraca punkt srodka grafiki/obiektu
getPktShipOgolny- zwraca punkt obiektu/grafiki na mapie
getIsEnemy- zwraca informacje czy pocisk nalezy do wroga
getObrazenia- zwraca ilosc obrazen danej broni
